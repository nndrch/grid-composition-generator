## Rules for the executing agent — read first

1. **Do only the step you were asked to do.** Do not touch code belonging to other steps.
2. **Match edits by content, not line number.** Line numbers below are correct as of writing but shift as steps land. Every edit gives the exact current code — find it verbatim.
3. **Read each file before editing it.**
4. **Never reformat, rename, or "improve" surrounding code.** These are surgical edits; diffs must contain only what this document specifies.
5. Architectural invariants (from `project-specification.md`) stay in force:
   - Zero runtime npm dependencies (Vite is dev-only).
   - Native APIs only: `Blob`, `URL.createObjectURL`, `XMLSerializer`, `<canvas>`, `navigator.clipboard`, `navigator.share`.
   - Exported SVG carries `viewBox` only — no `width`/`height`.
6. **Verify before committing**: run `npm run dev`, open http://localhost:5173, walk the step's checklist. All boxes must pass. Then run the step's grep checks — they must return nothing (or exactly what the step says).
7. Commit with the exact message given in the step. Do not push unless asked.
8. If anything doesn't match this document (code not found, unexpected extra references), **stop and report** — do not improvise.
